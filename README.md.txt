# Event assignment using observable notebook

The notebook is located [here](https://beta.observablehq.com/d/a94651ff4bb1a056)
The text cells are currently written in finnish, english versions coming.

Uses modified algorith from [Li et al. (2014)](https://www.cs.ucsb.edu/~klee/papers/On_Social_Event_Organization_papers.pdf)

The algorithm is modified to handle groups that have same preference for attendance. No community awareness is considered yet.
